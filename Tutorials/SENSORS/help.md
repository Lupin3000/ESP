# Sensor Help

## IR Flame Sensor

### Necessary parts

- 1x LED (_any color_)
- 1x Resistor (_min. 220 ohms_)
- 1x Flame Sensor (_3 Pin Digital_)

## Light/Shadow detection

### Necessary parts

- 1x LED (_any color_)
- 1x Resistor (_min. 220 ohms_)
- 1x Photo resistor
- 1x Resistor (_min. 10 kilo ohms_)
